WordPress Website Creation

For Prac 3, I created a WordPress website for my startup idea called MindMate AI. I used CloudAccess free hosting to install WordPress.

After installation, the website showed the default “Hello World” post and a sample page. I deleted those and created my own pages similar to what I made in Joomla for Prac 2.

The pages I created are:

Home

Features

Team

Contact

I also changed the homepage from the default blog page to a static page (Home) using Settings → Reading. Then I fixed the menu order so it shows:

Home → Features → Team → Contact

This made the website look more organised and professional.

Customisation

I edited the content so it matches my startup idea:

The Home page explains what MindMate AI is.

The Features page explains chat support, breathing exercises, mood tracking, etc.

The Team page introduces the founder and team members.

The Contact page includes email and phone details.

Compared to Joomla, I found WordPress easier to use. The dashboard is simple and clean. However, the menu editing in the new theme was a bit confusing at first.

Joomla vs WordPress (My Opinion)

Joomla:

More structured

Slightly more technical

Takes more time to understand

WordPress:

Easier for beginners

Faster to set up

Simpler page editing

Overall, I personally found WordPress easier to manage.

Group Interview Activity Reflection

In class, we did a short “interview style” activity to help choose group members for the major project.

I asked questions like:

What is your availability outside class?

Do you usually complete work on time?

What are your strengths?

Have you used CMS platforms before?

From this activity, I realised that a good group member is not only someone with strong technical skills, but also someone who:

Communicates clearly

Is responsible

Meets deadlines

Contributes equally

Is serious about the project

This activity helped me think more carefully about choosing teammates.
